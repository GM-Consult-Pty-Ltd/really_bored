Bored

Bored class Null safety

A simple implementation of The Bored API.

See https://www.boredapi.com/documentation

A sample JSON response body from the API follows below.

  { 'activity': 'Learn a new programming language',
    'type': 'education',
    'participants': 1,
    'price': 0.1,
    'link': '',
    'key': '5881028',
    'accessibility': 0.25 }
